jQuery colourLoader plugin

A simple demonstration of the jQuery colourLoader plugin - random colours.

View the rainbow colours or custom colours options.

Sample code:

$(document).ready(function() {
	//initialize colourLoader
	$('body').colourLoader("random");
	//fade out when page is done loading
	$(window).load(function() {
		$('.loader').fadeOut(300).css('display', 'none').stop();
	});
})
